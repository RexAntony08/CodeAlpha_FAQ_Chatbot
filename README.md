# 🤖 CodeAlpha FAQ Chatbot

## 📌 Project Overview

This project is a simple AI-powered FAQ Chatbot developed as part of the **CodeAlpha Python Programming Internship**. The chatbot answers user queries by matching them with a predefined set of Frequently Asked Questions (FAQs) using Natural Language Processing (NLP) techniques.

## ✨ Features

* Stores FAQs in a JSON file.
* Preprocesses text using the **NLTK** library.
* Uses **TF-IDF Vectorization** and **Cosine Similarity** to find the most relevant answer.
* Interactive command-line chatbot interface.
* Easy to customize by adding more FAQs.

## 🛠️ Technologies Used

* Python 3
* NLTK
* Scikit-learn
* JSON

## 🚀 Installation

1. Clone the repository:

   ```
   git clone https://github.com/RexAntony08/CodeAlpha_FAQ_Chatbot.git
   ```
2. Navigate to the project folder:

   ```
   cd CodeAlpha_FAQ_Chatbot
   ```
3. Install the required libraries:

   ```
   pip install -r requirements.txt
   ```

## ▶️ How to Run

Run the following command in the terminal:

```
python faq_chatbot.py
```

The chatbot will start, and you can ask questions related to the stored FAQs. Type `exit` to quit the chatbot.

## 💬 Example

**You:** How can I apply for admission?
**Bot:** You can apply online through the college admission portal.

**You:** What documents are required for admission?
**Bot:** You need 10th and 12th mark sheets, ID proof, and passport-size photographs.

## 🎯 Project Objective

The objective of this project is to demonstrate the use of basic NLP techniques and similarity matching to build an intelligent FAQ chatbot that can answer user queries efficiently.
